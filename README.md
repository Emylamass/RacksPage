# 🚀 Racks Agência — Portfólio Web

Este projeto é um site moderno e responsivo desenvolvido para apresentar serviços criativos como design, social media, videomaker e publicidade.

O objetivo é transmitir profissionalismo, criatividade e facilitar o contato com clientes.

---

## 🎨 Tecnologias Utilizadas

- HTML5  
- CSS3 (Flexbox + Grid + Responsivo)  
- JavaScript  

---

## ⚙️ Funcionalidades

✔ Menu responsivo com hamburger  
✔ Scroll dinâmico no header  
✔ Botão de WhatsApp clicável  
✔ Botão flutuante fixo  
✔ Layout moderno e profissional  
✔ Design responsivo (mobile e desktop)  
✔ SEO básico implementado
